# Fashion-recommendation-system
![Python](https://img.shields.io/badge/python-3.8%2B-blue)
![TensorFlow](https://img.shields.io/badge/TensorFlow-2.8%2B-orange)
![License](https://img.shields.io/badge/license-MIT-green)

A deep learning-based fashion recommendation system that finds similar clothing items using visual features extracted from images.

##  Features

- **Image Feature Extraction** using ResNet50
- **Similarity Search** using cosine similarity
- **Visual Recommendations** with similarity scores
- **Interactive Demo** in Jupyter Notebook
- **Easy-to-use API** for integration
